## Utility Types

TypeScript provides several utility types to facilitate common type transformations. These utilities are available globally.

```javascript
Partial<T>
Readonly<T>
Record<K,T>
Pick<T,K>
Omit<T,K>
Exclude<T,U>
Extract<T,U>
NonNullable<T>
ReturnType<T>
InstanceType<T>
Required<T>
ThisType<T>
```
